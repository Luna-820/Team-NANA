# Tag → Class Mapping

| Tag  | Class            | Role / Usage                       | Visual Weight / Size|
|------|------------------|------------------------------------|---------------------|
| `h2` | `.section-title` | セクションの大見出し、ページ構成の区切り  | 大 / Bold           
| `h3` | `.box-title`     | カード・ブロック・まとまりの単位の見出し  | 中 / Bold           
| `h4` | `.message`       | 小さな補助見出し、導入・前置きのニュアンス | 中 / Medium         
| `p`  | `.text`          | ベースの本文、最も読む量が多いテキスト    | 標準 / Light       
| `p`  | `.middle`        | 想いやメッセージを強調したい短文         | 少し大きい / Medium  
| `p ` | `.small`         | 注釈、補足、小さく添える説明            | 小さい / Light       

# その他フォント関係のメモ
$lh-heading	見出しの行間 (line-height for heading)
$lh-body	本文の行間 (line-height for body)
$ls-base	字間 (line-height for base)
